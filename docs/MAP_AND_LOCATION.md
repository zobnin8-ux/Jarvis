# Карта и геолокация (Jarvis)

> **Решение (2026):** для личного HUD **оставляем OpenStreetMap**. Google Maps — **опционально позже**, не критично.

---

## Сейчас в продукте

| Что | Как |
|-----|-----|
| **Карта в футере** | Leaflet + тайлы [OpenStreetMap](https://www.openstreetmap.org/) (Carto/OSM) |
| **Геолокация устройства** | `navigator.geolocation` → одна точка на карту и погоду |
| **Название места** | `/api/geocode` → Nominatim (+ Open-Meteo fallback), **не** ISS-океанский текст |
| **HOME fallback** | `WEATHER_LAT` / `WEATHER_LON` / `WEATHER_CITY` в `.env.local` |
| **Метки** | **ЗДЕСЬ** (live GPS/Wi‑Fi) · **HOME** (fallback) |
| **Управление картой** | pan (drag), zoom (колесо, +/−, двойной клик), ◎ — к метке, OSM — полная карта |

**Файлы:** `LocationMapModule.tsx`, `DeviceLocationContext.tsx`, `deviceGeocode.ts`, `api/geocode/route.ts`

**Стоимость:** **$0** — OSM/Nominatim для личного использования без billing.

## Google Maps — если когда‑нибудь понадобится

Не обязателен. Имеет смысл, если нужны спутник, привычный Google UI или их POI/маршруты.

### Когда начнут списывать (с ~марта 2025)

Billing-аккаунт и карта **нужны**, но **платёж только после превышения бесплатного лимита по SKU**:

| SKU | Бесплатно / мес | После лимита (ориентир) |
|-----|-----------------|-------------------------|
| **Dynamic Maps** (JS-карта) | **10 000** загрузок | ~$7 / 1000 |
| Static Maps | 10 000 | ~$2 / 1000 |
| Geocoding | 10 000 | ~$5 / 1000 |
| Map Tiles API | 100 000 | по объёму |

Официально: [Google Maps Platform pricing](https://developers.google.com/maps/billing-and-pricing/overview)

**1 загрузка ≈ 1 открытие Jarvis с картой** (refresh ≈ +1).

### Jarvis «на тик», иногда

| Частота | Загрузок/мес | Счёт |
|---------|--------------|------|
| 2–3 раза в неделю | ~10–15 | **$0** |
| Раз в день | ~30 | **$0** |
| 10 раз в день | ~300 | **$0** |
| ~10 000+ открытий/мес | — | начинается платный tier |

Для личного редкого запуска **неожиданного списания почти не будет**, если ключ не утёк и не включены лишние API.

### «Неожиданные» деньги — обычно из‑за

- API-ключ в открытом доступе (боты)
- ключ без ограничений по referrer/IP
- Places / Directions / Street View без нужды
- баг (карта перезагружается в цикле)
- нет budget alert в GCP

### Минимальная безопасная настройка Google (если перейдём)

1. Включить **только** Maps JavaScript API (+ Geocoding при необходимости).
2. Ключ: `http://localhost:3001/*` + домен/IP дома.
3. **Budget alert** $1–5 в [Google Cloud Billing](https://console.cloud.google.com/billing).
4. Опционально: дневная квота на API.
5. Env: `NEXT_PUBLIC_GOOGLE_MAPS_KEY` — **не коммитить**.

## Связанное

- Погода использует те же `lat/lon` из `DeviceLocationProvider`.
- ISS в футере — **отдельная** орбитальная геолокация (`issGeocode.ts`), не путать с «ЗДЕСЬ».
